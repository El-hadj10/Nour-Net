## Projet phare — Nour-Net

Moteur d'exploration et de validation de cibles reseau avec Command Center local.

**Ce que ca fait :**
- Scraping Brave Search via tunnel Tor (dorks configures)
- Filtre de pertinence : elimination du bruit avant validation
- Validation HTTP HEAD/GET de chaque cible
- Persistance dedupliquee dans `botnet/zombies.txt`
- Interface GUI FastAPI : stream WebSocket, carte Leaflet, export JSON/CSV

**Stack :** Python · FastAPI · Uvicorn · websockets · BeautifulSoup4 · Leaflet · Brave Search · Tor/Privoxy
